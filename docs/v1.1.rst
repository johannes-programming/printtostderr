Overview
--------

This project provides a function that prints to ``sys.stderr``.

Installation
------------

To install ``printtostderr``, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

    pip install printtostderr

Typing
------

This project is strictly typed with ``mypy``.

.. container:: versionadded

    **Added in version 1.1.**

Features
--------

``printtostderr.main(args: Optional[Iterable[str]] = None) -> None``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function parses the ``args`` and hands them to the ``printtostderr`` function.

.. container:: versionchanged

    **Changed in version 1.1:** ``args`` is now typed as ``Optional[Iterable[str]]`` instead of ``Optional[Iterable]``.

``printtostderr.printtostderr(*args: object, **kwargs: Any) -> None``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function prints to ``sys.stderr``.

.. code-block:: python

    def printtostderr(*args: object, **kwargs: Any) -> None:
        "This function prints to sys.stderr."
        print(*args, **kwargs, file=sys.stderr)

.. container:: versionchanged

    **Changed in version 1.1:** ``*args`` is now typed as ``object`` instead of ``Any``.

``printtostderr.tests.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its ``test`` function.

.. code-block:: python

    printtostderr.tests.test()
